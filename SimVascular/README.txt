simvascular_10_script - python script to generate meshes, presolver files and solver.inp files for 100 SimVascular cases
job1.job - example CHPC slurm file to run 10 SimVascular simulations

stenosis_geometries - the 10 stenosis geometries in stl format, [cm] units. Blockage ratio varies from 40% to 85% by increments of 5%
waveforms - 10 waveforms used as inlet boundary conditions (8 of them are the same as in the NeurIPS paper)
